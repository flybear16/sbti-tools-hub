# PRD v1.1：站内测评答题功能

> 版本：0.2（草案）
> 作者：增长运营官
> 日期：2026-04-12
> 状态：待评审

---

## 1. 背景与问题

### 1.1 v1.0 现状
- SBTI 测评工具导航站已上线（列表页 + 详情页）
- 用户点击"立即测试"后跳转至外部测评工具的原始页面
- 数据和用户体验完全依赖第三方平台

### 1.2 为什么做 v1.1
- **用户流失不可控**：跳转导致用户直接离开本站，站内停留时间为零
- **数据空白**：无法获知用户搜索、筛选了哪些测评，以及答题中途流失率等关键行为
- **商业价值低**：纯导航站广告变现天花板明显，自有测评数据才可做用户画像、社群运营、付费报告等后续转化
- **竞品差距**：头部测评聚合站均有完整站内答题+报告

---

## 2. 用户故事

```
作为：普通测评用户
我想要：在站内直接完成测评并看到结果
以便：不需要跳转到其他网站，且能方便地分享我的结果
```

**验收标准：**
- [ ] 从详情页点击"立即测试"后，在站内开始答题
- [ ] 答题过程不丢失进度（刷新后继续）
- [ ] 提交后立即看到人格类型和简要报告
- [ ] 可以一键分享到微信/朋友圈

---

## 3. 功能范围

### MVP（P0）
| 功能 | 说明 |
|------|------|
| 站内答题页 | 展示题目和选项，支持单选，实时保存进度 |
| 进度指示 | 显示当前第 N 题 / 共 M 题 |
| 上一题/下一题 | 支持前后翻题 |
| 结果页 | 显示人格类型 + 简要文字报告（约200字） |
| 分享链接 | 一键复制分享链接 |

### 后续迭代（P1+）
| 功能 | 说明 |
|------|------|
| 分享图片 | 结果页生成分享图（带人格类型+简要报告） |
| 历史记录 | 保留最近5次测评结果 |
| 题目多选 | 支持多选题型 |

### 不在本次范围
- 深度报告（付费墙）
- 用户账号体系
- AI 个性化分析
- 站内社群/评论

---

## 4. 题目来源

| 方案 | 说明 | 优点 | 缺点 |
|------|------|------|------|
| A：自建题目库 | 初期 2~3 个高频测评用 JSON 配置 | 完全可控、可商业化 | 建设周期长 |
| B：对接外部 API | 接入测评 API 提供商 | 省开发成本 | 有抽佣、数据不自有 |

---

## 5. 成功指标

| 指标 | 目标 |
|------|------|
| 站内答题完成率 | ≥ 60% |
| 平均站内停留时间 | ≥ 3 分钟/用户 |
| 结果页分享率 | ≥ 10% |
